# Sync-DSIndex

## SYNOPSIS
Takes one or more folders as input and runs content extraction and indexing over it/them.
If no folder is specified, already indexed folders are re-indexed.

## SYNTAX

```
Sync-DSIndex [[-Folder] <String[]>]
```
